2. Vue
3. VueX

### 功能
1. 建立、登入帳號密碼(賦予token -> Local Storage)
2. 修改個人資訊(右上)
3. 大廳聊天室
4. 建立房間(可賦予密碼)
5. 進入遊戲(遊戲內容為自創)
6. 結束畫面(自動刪除遊戲房間)

### 可優化範圍
1. 遊戲卡頓
2. 房間內偶爾會出現BUG需重新整理
3. 安全性
